СТРУКТУРА ПО:

Данная программа состоит из сервиса(папка service) и клиента (папка client)

ЗАПУСК:

Одновременно должны быть запущены два файла: service.py(service/service.py) и client.py(client/client.py). Сначала запускаете сервер, а затем уже клиент.Команду uvicorn не нужно писать в командной строки или терминале, она уже запуститься в качестве команды внутри файла service.py.
